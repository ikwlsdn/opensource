-  **sed 및 awk 명령어**

    - sed : 비대화형 편집기이며 하나의 입력이 하나의 출력을 내보낸다.
    
        1)정규표현식을 사용하기 때문에 특수문자앞에 역 슬래시(\)를 붙여주어야 한다.
        
        2)e를 사용하면 출력되는 값을 보여주기 때문에 sed -e의 형식으로 사용된다.
        
        |옵션|실행|
        |----|----|
        |-p |출력|
        |-d |삭제|
        |-s|치환|
        |,|범위 지정|
        |-e|다중 편집|
        |-r|파일에서 읽기|
        |-w|파일에 쓰기|
        |-a|추가|
        |-i|삽입|
        |-c|변경|
        |-n|다음|
        |-y|변환|
        |-q|종료
        
        
    - awk : 대화형 편집기, 입력 및 출력이 하나로 이루어진다.
